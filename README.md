Membuat eksternal style sheet dengan nama style.css. Informasi yang dapat Anda gunakan untuk membuat tampilan seperti pada file hasil_tampilan_yang_diinginkan.png adalah sebagai berikut:
- Font yang digunakan dalam keseluruhan teks adalah 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif
- Warna tulisan utama adalah dengan kode warna #2d2d2d
- Biru utama yang digunakan adalah kode warna #0046be
- Light Gray yang digunakan adalah kode warna #f6f8fb
- Dark Gray yang digunakan adalah kode warna #6c757d

Atur kode di style.css dengan ketentuan sebagai berikut:
- Tag <header> memiliki background dengan warna Light Gray dan padding atas-bawah 60px dan kiri-kanan 20px serta semua tulisan di dalamnya tengah (center). 
- Gambar yang ada di tag <header> memiliki lebar 260px dan tinggi 120px.
- Semua class btn-primary menggunakan warna background biru utama, tulisan berwarna putih, padding atas-bawah 12px padding kiri-kanan 30px, tanpa border, melengkung sebanyak 5px, size tulisan 1rem, cursor pointer.
- Gambar yang ada di tag <header> diterapkan animasi dengan keyframe bounce 2s dengan ketentuan keyframe 40% memanggil translateY(-30px), dan 60% memanggil translateY(-10px)
- Saat class btn-primary dilewati oleh cursor, maka warna akan berubah menjadi kode warna #00a6ff
- Tambahkan pengaturan styling pada style.css agar semirip mungkin dengan apa yang ditampilkan pada gambar hasil_tampilan_yang_diinginkan.png
